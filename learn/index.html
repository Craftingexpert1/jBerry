The basic function for jBerry is
jBerry(selector(querySelector string)).

You can run jBerry("body") and it will return an array of objects. It only has one at the moment, since there is (most likely) only one body. To get something from the array of objects, and in our case, the body, we can do jBerry("body").get(0). It will return the first object in the array, which is the body. Inspect the page and navigate to the console to try this.

There are lots of things you can do with jBerry. There is:

d.get(number): Gets a specific object out of an array of the chosen,
d.hide(): Hides the object,
d.show(): Shows the object,
d.tgl(): If hidden, shows the object, if shown, hides the object,
d.outhtml(string): Edits the outerHTML of object,
d.html(string): Edits the innerHTML of object,
d.text(string): Edits the text of object,
d.clsAdd(string): Adds a class to object,
d.clsRmv(string): Removes a class from object,
d.clsTgl(string): Adds/removes a class to/from object,
d.attr(name_string, value_string): Edits the attribute of an object: following name_tring=value_tring,
d.id(string):Edits the id of an object,
d.clk(function(){}):Makes an event listener for click,
d.fcs(function(){}):Makes an event listener for focus,
d.blur(function(){}):Makes an event listener for blur,
d.mseover(function(){}):Makes an event listener for mouse over,
d.msedown(function(){}):Makes an event listener for mouse down,
d.mseup(function(){}):Makes an event listener for mouse up,
d.msedbl(function(){}):Makes an event listener for double click,
d.mseenter(function(){}):Makes an event listener for mouse enter,
d.mseright(function(){}):Makes an event listener for context menu (the menu that pops open when you right click),
d.mseleave(function(){}):Makes an event listener for mouse leave,
d.msemove(function(){}):Makes an event listener for mouse move,
d.mseout(function(){}):Makes an event listener for blur,
d.keypress(function(){}):Makes an event listener for keypress,
d.style(string):Edits the style attribute,
d.bg(string):Edits the background,
d.color(string):Edits the text color,
d.border(string):Edits the border,
d.transform(string):Edits the transform,
d.transl(axis, howmuch):Edits the translate,on the axis you pick and howmuch you want it to translate,
d.rot(axis, howmuch):Edits the rotate, on the axis you pick and howmuch you want it to rotate,
d.scl(axis, howmuch):Edits the scale, on the axis you pick and howmuch you want it to scale,
d.skew(axis, howmuch):Edits the skew, on the axis you pick and howmuch you want it to skew,
d.persp(string):Edits the perspective value,
d.anim(string):Edits the animation property,
